Veil of the Hearthstar

Original fantasy action-adventure vertical slice: Hearthmere Fields

Hearts: ♥ ♥ ♥
Hearthstar Shards: 0 / 3
Objective: Recover shards to open the Ember Gate

Controls

  • WASD or Arrow Keys: move
  • Space: light attack
  • F: heavy attack
  • Shift: dodge
  • Q: parry window
  • L: lock-on target
  • E: interact / talk
  • K: save checkpoint
  • P: pause
  • R: restart

Lantern smoke drifts across the fields.

Slice Features

  • Combat + parry + dodge foundation
  • Lock-on targeting
  • Quest, inventory, and save scaffolding
  • Boss framework with phase transition
